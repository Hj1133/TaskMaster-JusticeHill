# TaskMaster

TaskMaster is a task management tool built using Python and Tkinter. It allows users to create, prioritize, and manage tasks effectively.

## Features
- Task entry form with priority levels (High, Medium, Low)
- Task management (add, edit, delete tasks)
- Due date tracking and task completion
- Task summary reports
